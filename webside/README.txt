-To use the webpage one has to be connected to fuseki.
-Run the java code and downloaded the newest version of "FindMyFitness.ttl". 
-Go to localhost:3030 and create a dataset with the name "ds".
-Uploade "FindMyFitness.ttl" to the dataset.
(It is important that the Sparql endpoint is "http://localhost:3030/ds/query")

There are three buttons on the webpage
"Søk"- This button will show a table with the result of the input.
"Fant ikke det du trengte? Søk igjen"- This button will show similar classes to the users input.
"Skriv ut query som tekst"- This button shows the sparql queries that will be used for finding the results.
